# License Command Center

171 software contracts. Daily personalized alerts. $2.2M exposure surfaced.

## Features
- Personalized HTML email per ITS contact (sees only their licenses)
- Power BI dashboard — 5 KPI cards, 90-day forecast
- Python web portal — filter, export CSV, click-row detail
- Smartsheet CSV import pipeline
- Full audit log
